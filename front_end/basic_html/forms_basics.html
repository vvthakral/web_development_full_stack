A very basic login form

This will not store data anywhere unless we connect something on the backend. We can have input inside or outside a form. If multiple input are related we can combine them using the form tag which is a container. By default action is same site and method is get.

The address bar has the action url followed by the ? with key and value pairs separated by &. For passwords we do not send a get request but do a post request so the password in reality does not shoe up in url.

We can add labels for information which can be accessed by visually impaired. They do not make any difference in the way the site appears but adds additional info for the screen readers. Also they can used to reference to any tag by mapping them to the id of the.
