Divs and spans

Divs and spans both are useful when we want to do stypling. They group elements and thus we can set properties for the grouped elements instead of setting a property for an individual element.

Divs: They are block level containers, so if we add a div inside a div it will break it into a new line. Pretty much how block level containers work. Didn't understand? See the para under div and compare to the same thing in span.

Spans: They are similar to the div except the fact that these are line level and not block level. So they do not break into new lines like how div did it in the above case. So why use div? Well, in the para graph like this div does not help but there will be instances where we want to break into new line but still have the same formatting, so div will be useful
